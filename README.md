## 关于本项目

这是 CFPA 团队 [Minecraft 模组简体中文翻译项目](https://github.com/CFPAOrg/Minecraft-Mod-Language-Package)的词典化整合，托管在 MC 百科的 [Minecraft 模组翻译参考词典](https://dict.mcmod.cn/)使用本仓库的数据。
